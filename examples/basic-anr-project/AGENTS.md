# AGENTS.md

## Purpose

This is a minimal ANR-compliant example project.
It demonstrates global and directory-level agent context.

## Repository structure

- `src/` application code
- `tests/` automated tests
- `tools/` scripts/utilities
- `docs/` documentation

## Agent context hierarchy

1. `AGENTS.md` (global context)
2. `*/AGENT.md` (directory context)
