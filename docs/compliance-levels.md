# ANR Compliance Levels

## Level 1 - Basic ANR

### MUST include

- `AGENTS.md`
- `.agents/context-index.md`

## Level 2 - Structured ANR

### MUST include

- all Level 1 requirements

### SHOULD include

- directory-level `AGENT.md` files
- `.agents/workflows/`

## Level 3 - Full ANR

### MUST include

- all Level 2 requirements

### SHOULD include

- `.agents/skills/`
- `.agents/guardrails/`
- `anr.yaml`
